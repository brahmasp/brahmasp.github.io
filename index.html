Publications
Reducing Sampling Error in Batch Temporal Difference Learning
Brahma S. Pavse
MS Thesis.
bibtex /

We propose PSEC-TD(0), an algorithm to correct for policy sampling error in batch TD(0). We conduct a theoretical analysis and empirical study showing that PSEC-TD(0) converges to a more accurate value function compared to TD(0) with respect to the true value function in this finite batch setting.
Reinforced Inverse Dynamics Modeling for Learning from a Single Observed Demonstration (RIDM)
Brahma S. Pavse, Faraz Torabi, Josiah Hanna, Garrett Warnell, Peter Stone
Undergraduate honors thesis.
ICML workshop 2019. Under review at IROS 2020.  
bibtex / video

We propose a novel reinforcement learning and imitation from observation algorithm that helps an agent learn from a single observed demonstration with no task-specific information encoded in the state space. We also show that the PID controller can be used as an effective inverse dynamics model.
